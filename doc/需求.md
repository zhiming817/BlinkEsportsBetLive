# 📝 产品需求文档 (PRD): BlinkBet

**版本：** v1.2 (Hackathon Version)
**状态：** 开发中
**目标：** 在 2026 Solana 黑客松中展示通过 Blink 进行极简赛事预测的闭环体验。

---

## 1. 产品概述
**BlinkBet** 是一个基于 Solana Actions & Blinks 的赛事预测平台。它允许用户直接在社交媒体（如 Twitter）或移动端 App 中，对未来的比赛结果进行下注。项目核心在于简化电竞预测流程，无需复杂的实时数据监听，重点实现“赛前下注，赛后自动结算”。

### 核心价值主张
* **零跳转下注：** 利用 Blink 技术在推特等平台实现原生下注体验。
* **异步结算：** 比赛结束后由系统自动对接数据源并触发链上资金分发。
* **低开发成本：** 专注于核心胜负预测，避开复杂的实时微预测逻辑。

---

## 2. 核心功能需求

### 2.1 赛事列表 (Match List)
* **未来预测：** 渲染即将进行的比赛卡片，显示战队信息、开赛时间。
* **下注入口：** 每个赛事卡片对应一个下注操作，支持胜/负双选。

### 2.2 预测交互 (Prediction Interaction)
* **下注详情：** 用户选择看好的战队，输入 SOL/USDC 金额进行确认。
* **链上提交：** 通过 Blink 在推特或 App 内唤起钱包完成交易。

### 2.3 赛后结算 (Post-Match Settlement)
* **状态监听：** 后端服务定时拉取 PandaScore 等数据源，检测比赛结果。
* **自动分款：** 确认结果后，后端调用合约结算接口，将奖金按比例池分发给胜方。

### 2.4 钱包与成就 (Wallet & Profile)
* **资产概览：** 显示余额、历史预测记录及总收益（Profit）。
* **通知提醒：** 结算完成后，用户在钱包或 App 内收到资金变动提醒。

## 3. 技术架构 (Technical Stack)

| 模块 | 技术选型 | 备注 |
| :--- | :--- | :--- |
| **移动端框架** | Expo / React Native | 适配 iOS/Android，支持深色模式 |
| **智能合约** | Anchor / Rust | 实现资金托管与分配逻辑 |
| **前端协议** | Solana Actions & Blinks | 符合最新的 Dialect 交互标准 |
| **数据源** | PandaScore API | 获取定时更新的比赛结果 |
| **任务调度** | BullMQ / GitHub Actions | 定时轮询比赛数据，触发结算 |

---

## 4. 业务流程 (User Flow)

1.  **分发：** 官方在 Twitter 发布下一场赛事的预测 Blink。
2.  **预测：** 用户在推特卡片内点击选择胜方并签名交易。
3.  **结果：** 比赛开始并结束。
4.  **核验：** 后端任务发现比赛结束，获取官方结果。
5.  **分账：** 后端签名触发智能合约，合约根据池子比例完成资金自动划转。
