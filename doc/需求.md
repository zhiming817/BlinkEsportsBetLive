# 📝 产品需求文档 (PRD): BlinkBet Live

**版本：** v1.1 (App Internal Beta)
**状态：** 开发中
**目标：** 在 2026 Solana 黑客松中展示 Web3 电竞预测的极致 App 体验。

---

## 1. 产品概述
**BlinkBet Live** 是一个基于 Solana Actions & Blinks 的垂直电竞预测协议与移动端应用。它允许用户通过移动端 App 或社交媒体交互卡片，对正在进行的电竞比赛（如 LoL, Dota 2, CS2）中的“微型事件”进行毫秒级下注与结算。

### 核心价值主张
* **多端一致体验：** 完美融合 Native App 的流畅度与 Blink 的传播力。
* **即时快感：** 针对“下一次团战”、“下一血”等短周期事件，结果即出，资金即回。
* **酷炫视觉语言：** 深色科技风 UI，实时动态赔率走势曲线。

---

## 2. 核心功能需求

### 2.1 仪表盘与实时对阵 (Dashboard)
* **Live 赛事追踪：** 实时渲染正在进行的比赛卡片，包含战队 Logo、实时时长及对阵信息。
* **实时赔率曲线 (Live Odds)：** 动态图表展示 T1/Gen.G 等战队随比赛进程变化的赔率走势，辅助用户决策。
* **微预测列表 (Micro-Predictions)：** 横向滚动展示“首血”、“下一塔”等高频预测项，支持一键进入详情。

### 2.2 预测详情与交互 (Prediction Detail)
* **情境化下注：** 根据特定预测题目（如：Who will get the First Blood?）提供双选或多选大按钮。
* **发光选中态：** 增强视觉反馈，选中后按钮呈现品牌青色发光效果。
* **奖池统计：** 实时同步 Total Pool（总奖池）与 Your Stake（当前下注额）。

### 2.3 自动结算与奖励 (Settlement)
* **实时结算通知：** 比赛事件达成后，App 立即弹出结算详情卡片。
* **成功激励：** 醒目的奖励数值显示（如 +0.5 SOL），以及基于 Token Extensions 的即时到账反馈。
* **闪电结算：** "Settled Instantly" 进度条展示技术底座带来的亚秒级响应。

### 2.4 钱包与资产管理 (Wallet)
* **多币种支持：** 直观展示 SOL, USDC, BLINK 代币余额及折算 USD 价值。
* **转账操作：** 集成 Deposit（充值）、Withdraw（提现）、Swap（兑换）快捷入口。
* **交易历史：** 记录包括下注支出、奖金收入、兑换变动在内的完整资金流水。

### 2.5 个人成就系统 (Profile)
* **数字化身份：** 赛博风格头像装饰，展示用户称号（如 Pro Predictor）。
* **战绩看板：** 实时计算胜率（Win Rate）及累计盈利（Total Profit）。
* **安全偏好：** 包含通知开关、双重身份验证（2FA）、关联钱包管理。

## 3. 技术架构 (Technical Stack)

| 模块 | 技术选型 | 备注 |
| :--- | :--- | :--- |
| **移动端框架** | Expo / React Native | 适配 iOS/Android，支持深色模式 |
| **智能合约** | Anchor / Rust | 部署在 Solana Mainnet |
| **组件库** | Custom SF Symbols | 兼容 Android 的 MaterialIcons 映射方案 |
| **路由管理** | Expo Router | 扁平化文件系统路由，支持 Tab 切换 |
| **状态存储** | **ZK Compression** | 用于存储海量用户的历史预测数据，降低租金成本 |
| **前端协议** | Solana Actions & Blinks | 符合最新的 Dialect 交互标准 |
| **数据源** | PandaScore API + Pyth | 获取电竞实时数据与代币价格 |

---

## 4. 业务流程 (User Flow)

1.  **分发：** 官方或 KOL 在推特发布 Blink，或用户直接打开 **BlinkBet Live App**。
2.  **预测：** 用户在首页点击感兴趣的赛事卡片，进入详情页选择预测项并确认。
3.  **上链：** 交易通过 Solana 网络执行，用户获得对应的预测凭证。
4.  **监测：** 实时赔率曲线随比赛波动更新。
5.  **结算：** 预测成功后，App 弹出结算卡片，展示奖励金额并自动更新钱包余额。

---

## 6. 关键指标 (KPIs)

* **UI 一致性：** App 内部页面与 Blink 卡片的视觉风格达成 100% 统一。
* **End-to-End Latency：** 从点击下注到链上状态变更（目标：<$500ms$）。
* **活跃留存：** 通过 Profile 胜率排名和盈利看板提升用户参与频次。
